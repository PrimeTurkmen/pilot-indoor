Indoor Positioning

PILOT Extension for indoor people and asset tracking

Purpose

Tracks people and assets inside buildings with ±1m accuracy using Bluetooth 6.0 Channel Sounding (nRF54L15).

  • Floor plan overlay on Leaflet map (CRS.Simple)
  • Real-time tag markers via Pilot API V3
  • Buildings → Floors → Zones navigation
  • Geozones, movement trails, heatmaps

Data Sources

Pilot API V3 — device positions, units. Positioning Engine publishes computed positions via API.

MQTT — Positioning Engine subscribes to pilot/indoor/distances/+ from gateways.

Configuration

Set devicesApiUrl in config.json to your positioning engine (e.g. http://your-engine:3080/api/indoor/devices) so the extension can:

  • Load floor plan — Select a floor, enter plan URL and bounds, click “Load on map”, then “Save to engine”.
  • Calibration — Edit 3-point pixel↔geo calibration and save to engine.
  • Anchors — Edit anchor positions per floor and save to engine.

Extension base URL: /store/indoor-positioning/.
